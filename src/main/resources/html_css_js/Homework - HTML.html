Setup Guide


Java SDK


1. Go to Java Website and download Java SE Development Kit 19

2. Install

3. Add the JDK path to system PATH enivornment variable

4. Test the installation with comand

java -version


Apache Maven


1. Go to Maven Website

2. Download the latest binary zip

3. Unzip to a directory of your choice

4. Add the bin dir of the installation to PATH

5. Create a new env variable called JAVA_HOME pointing to the JDK path

6. Test the Maven intsallation with command

mvn -version


intelliJ IDEA


1. Go to IntelliJ Website

2. Download IntelliJ IDEA Community Edition

3. Install


GIT


1. Go to git Website

2. Create a new user environment variable called HOME, pointing to your user’s home directory (C:\Users\username)

3. Add the GIT install path to PATH environment variable

4. Test the GIT installation with command

git

5. Configure GIT with commands

git gitconfig--global user.name “your user”

git gitconfig--global user.email“your email”


GIT repository


1. Go to gitlab Website

2. Register

3. Create a new repository (project)

4. Give developer permissions to my username (admin.xxx)

git

5. Configure GIT with commands

git gitconfig--global user.name “your user”

git gitconfig--global user.email“your email”
